ESTO ES UNA PRUEBA DE GIT

Vamos a hacer un Commit

  • Primero se crear la carpeta donde vamos a trabajar en local.
  • Después se abre el shell de git en esa carpeta
  • Ahora hacemos un git init para inicializar. SOlo se hace una vez por proyecto
  • Creamos nuestros archivos, modificamos, etc...
  • Antes de hacer un commit, siempre hay que hacer un git add . Este comando envía al stage area todos los archivos del proyecto.
  • Por últimoi se hace un git commit para salvar los cambios realizados.
